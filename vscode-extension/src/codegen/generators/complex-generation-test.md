# Генерация сложного графа с ветвлениями

## Описание теста

Тест проверяет генерацию C++ кода для сложного графа с:
1. Начальным узлом
2. Узлом ветвления с условием
3. Двумя узлами в ветках True/False  
4. Тремя узлами вывода
5. Конечным узлом

## Ожидаемая структура кода

```cpp
// multicode:begin
#include <iostream>

int main() {
    // Узел: Начало
    std::cout << "Начало" << std::endl;
    
    // Узел: Ветвление (if (x > 5))
    if (x > 5) {
        // Узел: ВетвлениеИстина (if (y > 10))
        if (y > 10) {
            // Узел: Вывод1
            std::cout << "Условие истинно" << std::endl;
        } else {
            // Узел: Вывод2
            std::cout << "Условие ложно" << std::endl;
        }
    }
    
    // Узел: Вложенное условие (if/else if/else)
    if (nested_condition) {
        std::cout << "Вложенное условие" << std::endl;
    }
    
    // Узел: Конец
    std::cout << "Конец" << std::endl;
    
    return 0;
}
// multicode:end
```

## Критерии проверки

1. ✅ Корректная генерация всех узлов
2. ✅ Правильная вложенность if/else структур  
3. ✅ Все выводы присутствуют в коде
4. ✅ Отсутствие синтаксических ошибок
5. ✅ Русские комментарии с названиями узлов
6. ✅ Валидный C++ синтаксис
7. ✅ Правильный топологический порядок

## Проблема для исследования

Текущая реализация кодогенератора может иметь проблемы с:
- Генерацией вложенных if/else конструкций
- Обработкой сложных топологических зависимостей
- Корректным определением областей видимости переменных
