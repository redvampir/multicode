# Скриншот: Blueprint-редактор (заглушка)

Бинарные изображения в репозиторий не добавляются.

## Что должно быть на реальном скриншоте
- Верхняя панель с режимом **Визуальный** и переключателем RU/EN.
- Канвас Blueprint-редактора с узлами: `Начало`, `Ветвление`, `Вывод`, `Конец`.
- Боковая панель переменных.

## Как сгенерировать актуальный скриншот локально
1. Открыть проект в VS Code.
2. Запустить Extension Development Host (`F5`).
3. Открыть MultiCode editor.
4. Сделать снимок экрана и сохранить вне репозитория или в релизные артефакты.
