# Icons for MultiCode Visual Nodes

This folder contains simple SVG icons intended to be used in the VS Code webview for node categories.

- control.svg — execution / flow control
- loop.svg — loop structures
- variable.svg — variables / storage
- math.svg — arithmetic operations
- logic.svg — logical operations
- io.svg — input/output

Usage: reference these files from the webview (e.g. `vscode-extension/media/icons/control.svg`).
