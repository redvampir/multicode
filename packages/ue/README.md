# @multicode/ue

UE-ориентированная библиотека узлов MultiCode.

## Границы ответственности

- Содержит только узлы, связанные с Unreal Engine/Blueprint-паттернами.
- Может использовать UE-типы портов (`pointer`, `class`) и UE-терминологию.
- Подключается отдельно от `@multicode/std` через настройку `multicode.packages.enableUe`.

## Политика релизов

- Версионирование: SemVer.
- Версия пакета независима от `@multicode/std`.
- Ломающее изменение сопровождается заметкой о миграции UE-графов.

## Совместимость

- При выключенном пакете UE-узлы не загружаются в реестр.
- Графы с UE-узлами должны деградировать корректно (узел неизвестного типа, без падения редактора).
