# Roadmap (реалистичная)

## v0.1 - «Граф можно сохранить и открыть» (целевой процент: 40%)
- [ ] `GraphSerializer` (JSON) + snapshot тесты.
- [ ] VS Code webview skeleton: добавление/удаление узлов, соединения, сохранение/загрузка файла.
- [ ] CLI/команда `Generate Code` возвращает заглушку текста.
- [ ] GitHub Actions: сборка core + запуск тестов.

## v0.2 - «Генерируем C++» (60%)
- [ ] `ICodeGenerator` + `CppCodeGenerator` с поддержкой:
  - Start/End, последовательность вызовов
  - Переменные (`Variable`, `SetVariable`, `GetVariable`)
  - `If`, `ForLoop`, базовые арифметические узлы
- [ ] Binding к исходникам: маркеры begin/end, запись файла.
- [ ] Пакеты узлов: loader + простейший пакет `std`.

## v0.3 - «Удобно использовать» (80%)
- [ ] Undo/Redo в редакторе.
- [ ] Предпросмотр кода в панеле.
- [ ] Базовые подсказки (автонаименование).
- [ ] Форматирование/линт в CI (clang-format, clang-tidy).

## Out of scope (до релиза 1.0)
- Совместная работа в реальном времени.
- Live-дебаг графов.
- Экспорт в другие IDE.

Каждый пункт помечаем `done`, только когда есть код и тесты в репозитории.
