# @multicode/std

Кросс-доменная библиотека узлов MultiCode.

## Границы ответственности

- Содержит только платформенно-нейтральные узлы (`flow`, `math`, `logic`, `io`, `string`, `array` и др.).
- Не содержит узлы, завязанные на Unreal Engine API и UE-термины.
- Может использоваться как базовый пакет по умолчанию для всех пользователей.

## Политика релизов

- Версионирование: SemVer.
- `MAJOR`: несовместимые изменения структуры узлов/портов.
- `MINOR`: новые обратносуместимые узлы/поля.
- `PATCH`: исправления без изменения контрактов.

## Совместимость

- Пакет не должен зависеть от `@multicode/ue`.
- Узлы графа из `std` обязаны открываться без дополнительных пакетов.
