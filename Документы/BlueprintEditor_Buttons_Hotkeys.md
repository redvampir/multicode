## ⌨️ Горячие клавиши

### Управление графом
- `Ctrl+Z` — Отменить последнее действие
- `Ctrl+Y` или `Ctrl+Shift+Z` — Повторить отменённое действие
- `F` — Вписать граф в viewport (Zoom to Fit)
- `L` — Применить автоматическую раскладку узлов

### Редактирование узлов
- `Ctrl+C` — Копировать выделенные узлы
- `Ctrl+X` — Вырезать выделенные узлы
- `Ctrl+V` — Вставить узлы из буфера
- `Ctrl+A` — Выделить все узлы
- `Delete` или `Backspace` — Удалить выделенные узлы

### Редактирование связей
- `Двойной клик` по связи — вставить контрольную точку (`Reroute`)
- `Alt + двойной клик` по связи — удалить связь

### Панели
- `A` — Открыть/закрыть палитру узлов
- `C` — Открыть/закрыть панель кода
- `P` — Открыть/закрыть менеджер пакетов
- `Escape` — Закрыть все панели и контекстное меню

> **⚠️ Важно:** Горячие клавиши `Ctrl+C`, `Ctrl+X`, `Ctrl+V`, `Ctrl+A` не работают когда фокус на поле ввода (input/textarea). Это сделано намеренно для работы с текстом.

## 🖱️ Контекстное меню

### Меню Canvas (правый клик на пустом месте)
- **Добавить узел** (`A`) — Открыть палитру
- **Вставить** (`Ctrl+V`) — Вставить из буфера
- **Отменить** (`Ctrl+Z`) — Undo
- **Повторить** (`Ctrl+Y`) — Redo
- **Выделить всё** (`Ctrl+A`) — Выделить все узлы
- **Вписать** (`F`) — Zoom to Fit
- **Автолейаут** (`L`) — Автоматическая раскладка

### Меню Node (правый клик на узел)
- **Копировать** (`Ctrl+C`) — Скопировать узел
- **Вырезать** (`Ctrl+X`) — Вырезать узел
- **Удалить** (`Delete`) — Удалить узел

## 📚 Палитра узлов

### Добавление узла
1. **Drag-and-drop:** Перетащить узел из палитры на canvas
2. **Клик:** Кликнуть по узлу — добавится в центр viewport
3. **Поиск:** Ввести текст в поле поиска для фильтрации

### Пользовательские функции
- Отображаются в категории "Function" под базовыми узлами
- Префикс **ƒ** перед названием
- Создают узел **CallUserFunction** для вызова функции

---

## 🔄 Система Undo/Redo

### Характеристики
- **Максимум истории:** 50 шагов
- **Debounce:** 500ms перед сохранением (группирует быстрые изменения)
- **Автоочистка:** История Redo очищается при новом действии

### Что запоминается
- Добавление/удаление узлов
- Перемещение узлов
- Добавление/удаление связей
- Вставка/перемещение контрольных точек связи (`Reroute`)
- Изменение свойств узлов

### Что НЕ запоминается
- Открытие/закрытие панелей
- Изменение zoom
- Выделение узлов

---

## 📋 Система Copy/Paste

### Особенности
- **Copy:** Копирует выделенные узлы + связи между ними
- **Cut:** Копирует + удаляет выделенные узлы
- **Paste:** Вставляет со смещением 50x50 пикселей
- **Повторная вставка:** Смещение удваивается (50, 100, 150...)
- **Новые ID:** Каждый вставленный узел получает уникальный ID
- **Автовыделение:** Вставленные узлы автоматически выделяются

---

## 🗂️ Переключение между графами

### EventGraph (основной граф)
- Вкладка **📊 EventGraph** — всегда доступна
- Содержит основную логику программы

### Функции
- Открывается через **FunctionListPanel** (кнопка ƒ)
- Каждая функция имеет свой граф узлов
- Вкладка **ƒ [Имя функции]** показывает активную функцию
- Клик по вкладке EventGraph возвращает к основному графу

## 🎨 Визуальная обратная связь

### Состояния кнопок
- **Active** — синяя подсветка (панель открыта)
- **Active-purple** — фиолетовая подсветка (панель функций)
- **Disabled** — серый цвет, cursor: not-allowed

- **Выделение** → синяя рамка вокруг узла

## 📝 Реализация (для разработчиков)

### Хуки
- **useUndoRedo** — История состояний с debounce
- **useClipboard** — Copy/Paste с генерацией новых ID
- **useAutoLayout** — Dagre/Klay алгоритмы раскладки
- **usePackageRegistry** — Реестр пакетов узлов

### Обработчики
- **handleUndo/handleRedo** — Синхронизация с historyState
- **handleCopy/handleCut/handlePaste** — Работа с clipboard хуком
- **handleDeleteSelected** — Удаление узлов + связанных связей
- **handleZoomToFit** — React Flow `fitView()`
- **handleAutoLayout** — Применение layout алгоритма
- **handleSelectAll** — Выделение всех nodes + edges

### Синхронизация
- **notifyGraphChange()** — Уведомляет родителя о всех изменениях
- **Debounce** — Откладывает уведомление через `setTimeout(..., 0)`
- **Dirty flag** — Устанавливается при любом изменении графа

---

## 🐛 Известные ограничения

1. **Вкладка активной функции** не реагирует на клик (стандартное UI поведение)
2. **Paste** работает только если есть данные в буфере (`clipboard.hasData()`)
3. **Undo/Redo** не работают для изменений, сделанных вне редактора
