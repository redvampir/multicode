# Тест Dropdown Selector для Переменных

## Цель
Проверить, что dropdown selector для указателей и классов работает корректно без черного экрана.

## Шаги для проверки

### 1. Создание переменных
1. Открыть файл `test.multicode` в VS Code
2. Создать переменную типа `pointer` (имя: `myPointer`)
3. Создать переменную типа `class` (имя: `myObject`)
4. Создать переменную типа `int` (имя: `counter`)

### 2. Проверка узла SetVariable
1. Добавить узел `SetVariable` на граф
2. Убедиться, что граф отрисовывается (НЕТ черного экрана) ✅
3. В узле SetVariable выбрать переменную типа `pointer` или `class`
4. Проверить, что появляется dropdown список доступных переменных
5. Выбрать переменную из dropdown
6. Убедиться, что выбор сохраняется

### 3. Проверка узла GetVariable
1. Добавить узел `GetVariable` на граф
2. Выбрать переменную типа `pointer` или `class`
3. Проверить, что dropdown работает аналогично

### 4. Проверка обычных переменных
1. Выбрать переменную типа `int` (не pointer, не class)
2. Убедиться, что dropdown НЕ появляется
3. Проверить, что обычный ввод работает как раньше

## Критерии успеха
- ✅ Граф отрисовывается нормально (НЕТ черного экрана)
- ✅ Dropdown появляется только для pointer/class типов
- ✅ Список переменных в dropdown корректный
- ✅ Выбор переменной работает
- ✅ Обычные переменные работают как раньше
